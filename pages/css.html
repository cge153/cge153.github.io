CSS stands for Cascading Style Sheets. The HTML elements get their style from CSS. This can be color, fonts, positioning and layout.

A webpage with plain HTML will be structured, but might still be difficult to read. CSS will make a webpage look good and ideally better to read.

Selectors

Universal Selector

  • Targets all elements

Type Selector

  • Matches the type of the HTML element
  • This is also referred to as the tag name selector or the element selector

Class Selector

  • Targets all elements with that class name
  • HTML elements can have multiple class values

ID Selector

  • The element ID must be unique on a given HTML document
  • Therefore, an ID targets only one specific element
  • Also, unlike to the class value, HTML elements can only have a single ID value

Attribute Selector

  • Targets a specific type of elements that have a specific attribute
  • Can be used in an even more specific way by targeting only specific attribut values
